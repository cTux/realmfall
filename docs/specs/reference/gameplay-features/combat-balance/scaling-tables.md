# Combat Balance Scaling Tables

## Scope

This reference lists the exact level `1` to `100` interpolation tables used by the current balance rules for items, players, and enemies.

## Item Rarity Rules

Item stat magnitudes use the same level `1` to `100` scaling for `common`, `uncommon`, `rare`, `epic`, and `legendary` items. Rarity changes secondary-slot capacity and generated secondary count, not the per-stat magnitude table.

| Rarity    | Secondary Slot Capacity | Min Generated Secondary Stats | Max Generated Secondary Stats |
| --------- | ----------------------- | ----------------------------- | ----------------------------- |
| Common    | 0                       | 0                             | 0                             |
| Uncommon  | 1                       | 0                             | 1                             |
| Rare      | 1                       | 0                             | 1                             |
| Epic      | 2                       | 1                             | 2                             |
| Legendary | 3                       | 1                             | 3                             |
